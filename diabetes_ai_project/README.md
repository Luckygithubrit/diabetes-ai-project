# AI Diabetes Prediction System

This is a simple healthcare AI project that predicts diabetes risk using Machine Learning.

## Technologies
- Python
- Scikit-learn
- Streamlit
- Pandas
- report lab

## Project Structure
diabetes_ai_project/
│
├── app.py
├── train_model.py
├── requirements.txt
│
├── dataset/
│   └── diabetes.csv
│
└── model/

## Setup Instructions

1. Install dependencies:
pip install -r requirements.txt

2. Train the model:
python train_model.py

3. Run the web app:
streamlit run app.py

Then open the browser at http://localhost:8501
